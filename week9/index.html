The window object

In web development, the window object is the global scope of the environment and it represents the browser window. Every time we declare a variable in the global scope it is assigned to window object. Example: x = 6; means x === window.x; //true

Some methods of the window object are dialogs:

  • alert() function will stop program executio and display a provided message in a dialog box, this methods returns undefined. Example: alert("Hello world")
  • confirm() function also will stop program execution and display a confirmation dialog with a provided mesasge and the OK or cancel options, ir returns true if OK is selected or false when Cancel is chosenlet go = confirm("De you want to continue?");
  • The prompt(). function stops execution of the program and displays a message provided also an input field for the user to enter text. When the OK button is clicked it will return the value of the input as string, if cancel is clicked then null will be returned: let userInput = prompt("Please enter a your name:")

Browser informatioon can also be obtained through the window object window.navigator. And information about the URL can be obtained using window.location like protoco, hostname, path, port and search (query string).

Browser history can be also be accessed and navigted:

window.history.length; // shows how many pages have been visited window.history.go(1); // goes forward 1 page window.history.go(0); // reloads the current page window.history.go(-1); // goes back 1 page

Windows

New windows can be created using the method window.open() that takes parameters:

  1. URL as parameter and
  2. Window name
  3. Attributes. some listed here
    • width: lists the CSS poperties that should be transitioned like: background, border and box model. transition-property: transforms
    • height: sets long the transition will take. it is specified in seconds: transition-duration: 0.2s
    • resizable: lets you control the pace of the transition in even more granular detail. Start off gast and end slower, or start off slow and get faster, key terms: ease, linear,ease-in, ease-out, or ease-in-out.

The window can be close using the method close(), modev using window.moveTo(0,0), or resized using window.resizeTo(600,400);

const popup = window.open('https://sitepoint.com','SitePoint','width=400,height=400,resizable=yes'); popup.moveTo(0,0); popup.resizeTo(600,400); popup.close();

HTML 5 API

HTML5 added a large number of new features to HTML and went beyond the actual markup language and brought together a number of related technologies such as CSS and JavaScript.

The data- attribute us a way of embedding daa in a web page using custom attributes ignored by the browser but accessible via JavaScript but they must use the following format:

  • Start with data-.
  • Contain only lowercase letters, numbers, hyphens, dots, colons or underscores.
  • Include an optional string value.
Each element has can be accessed in javascrip using the dataset property.

// HTML <div id='hero' data-powers='flight superSpeed'> Superman </div> // Javascript: const superman = document.getElementById('hero'); const powers = superman.dataset.powers;

Web storage API provides a key-value store on the client’s computer, this has fewer restrictions than cookies, more storage capacity, and is generally easier to use, the main differences with cookies are:

  • Information stored is not shared with the server on every request.
  • Information is available in multiple windows of the browser (but only if the domain is the same).
  • Storage capacity limit is much larger than the 4KB limit for cookies
  • Any data stored does not automatically expire as it does with cookies. This potentially makes cookies a better choice for something like showing a popup once a day
