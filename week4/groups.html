Groups

The Group class allows to add or remove items with using methods, it also contains the a static to create an instance of the class from an array or list of elements

Please see the console for results, this code will be executed:

let group = Group.from([10, 20]);
console.log(group.has(10));
// → true
console.log(group.has(30));
// → false
group.add(10);
group.delete(10);
console.log(group.has(10));
// → false
