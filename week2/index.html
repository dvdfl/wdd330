Notes:

Programming basics

Primitive data types are:

  • String
  • Symbol
  • Number
  • Boolean

Variables are defined using 'let' and constants using 'const', they are scoped. Variable names can begin with letter, undescore or $ sign.

Primitive type variables get assigned values, non-primitive are assigned by reference

Arrays, Logic, and Loops

Array is basic data structure, Set are list of values with additional methods

Logic control statements:

  • if
  • else
  • ternary (?)
  • switch

Loops are iteration (repeating) statements:

  • while
  • do... while
  • for

Functions

There are multiple was to define a function:

  • Function declaration
  • Function Expressions
  • Arrow Functions

  3. function that accepts a number as a parameter and check the number is prime or not
