Forms

HTML Forms contain controls that allow users to entere information to be processsed once the form has been submited.

Form controls have methods and properties that allow to:

  • Access the entered or selected values
  • Set a default value
  • Disable or enable the control.
  • Events can be used to add validations or restrictions.

Object Oriented Programming (OOP)

OOP is a style of programming on which the code is grouped in objects with methods and properties. The 3 principles of OOP are

  • Encapsulation
  • Polymorphism
  • Inheritance
