This

this is a keyword whose value changes depending on how a function gets called. Six differnt ways are

  • In global context. outside of any function it can referr the window object
  • In object construction. Here it references the instance of the object
  • In an object method. Also, refers to the instance.
  • In a simple function. Referrs the window object
  • In an arrow function. References the object
  • In an event listener. It is set to the element that fired the event in an event listener:

We can change the this context using bind method (not for arrow functions)

Objects

Objects are self-contained sets of related values or functions. the act as a collection of named properites

Two methods to create objects are:

  • literal object definition: const obj = {};
  • using constructor const obj = new Object();

Objects are copied by reference.

Literal objects are preferred for shorter definition. ES6 provides shorthand method where property key is same as variable name. They can also be used as parameters function greet ({greeting, name, age}) { alert("`${greeting} ${name}`")}

We can use prop in obj or hasOwnProperty(key) method to check if property exists on object. we can also delete a property using delete method: delete obj.property

JSON string is just like literal object with few diferences.

Math object provides values of constants and methods math operations.

Document Object Model (DOM)

The Document Object Model (DOM) allows to access elements of a web page and enable interation like adding or removing elements, their content and events

Node objects have type accessed using nodeType property, most common types are :

  • 1 - element
  • 2 - attribute
  • 3 - text
  • 8 - comment
  • 9 - body

Node lists are not arrays although the can be accessed using index but they can be turned into Array using the method Array.from(.

Events

Events are another part of the DOM are the link between web page and user interactions from:

  • Mouse
  • Keyboard
  • Touch
